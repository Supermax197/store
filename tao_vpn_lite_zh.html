道VPN Lite版可以在公用网络上建立专用网络，进行加密通讯，让用户进行远程访问。

道VPN Lite版是用Java开发的，跨几乎所有平台的，非常非常非常简单易用的VPN软件。无需配置、一键部署。麻雀虽小，五脏俱全。

道VPN Lite版支持http与https协议。也就是说支持几乎所有网站、网络游戏及应用。

道VPN Lite版用Java语言开发，这意味着几乎所有平台都可以运行，如windows、linux、mac os、android等，并且客户端、服务端可以运行在不同平台上。

道VPN Lite版有自己的独特的通信加密系统，因此与传统VPN软件并不兼容，这也使得道VPN具有独特的网络安全性与穿透性。

道VPN Lite版的使用非常简单。下载源代码编译或者直接下载jar包，启动客户端的命令：
java -cp tao-vpn-lite-1.0.jar org.tao.vnp.lite.client.Client xxx.xxx.xxx.xxx 8082 8081
,其中xxx是vpn服务端的ip地址，8082是服务端的端口号，8081是vpn客户端的端口号。启动服务端的命令：
java -cp tao-vpn-lite-1.0.jar org.tao.vpn.lite.Server 8082
,其中8082是服务端的端口号

在linux服务器上，可以使用类似的守护shell来启动服务端，
#! /bin/sh
while true ; do
NUM=`ps aux | grep -w tao-vpn-lite-1.0.jar | grep -v grep |wc -l`
if [ "${NUM}" -lt "1" ];then
echo "tao-vpn-lite-1.0.jar was killed"
nohup java -cp tao-vpn-lite-1.0.jar org.tao.vpn.lite.Server 8082 >/dev/null 2>%261 %26 fi
sleep 5s
done
exit 0

客户端和服务端都启动后，需要设置本地网络代理指向客户端的ip地址和端口，注意，只有vpn客户端到vpn服务端之间的通信是加密的，因此vpn客户端最好运行在本机上。
